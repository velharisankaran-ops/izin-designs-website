# IZIN Designs Landing Page Plugin

This WordPress plugin adds the shortcode:

```text
[izin_designs_landing]
```

Use it inside an Elementor Shortcode widget or a normal WordPress page.

## Safe Deployment

This plugin does not edit:

- WordPress core files
- Active theme PHP files
- Elementor plugin files
- `wp-config.php`
- `.htaccess`

Upload the plugin folder or ZIP to:

```text
wp-content/plugins/izin-designs-landing
```

Then activate **IZIN Designs Landing Page** from WordPress Admin > Plugins.

## Elementor Use

1. Create a new WordPress page.
2. Edit with Elementor.
3. Add a Shortcode widget.
4. Paste `[izin_designs_landing]`.
5. Publish the page.

This keeps the original WordPress website intact.
